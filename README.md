# Kaiten List Changer

Пользовательский скрипт (userscript) для браузера, позволяющий менять порядок полей в карточке **Kaiten** с помощью drag-and-drop. Порядок полей сохраняется между сессиями.

## Возможности

- **Drag-and-drop** — перетаскивайте поля за заголовки (`h6`), чтобы изменить порядок.
- **Автосохранение** — выбранный порядок полей запоминается и применяется при следующем открытии карточки.
- **Работа с динамическим контентом** — скрипт корректно обрабатывает SPA-переходы и открытие разных карточек без перезагрузки страницы.

## Требования

- Браузер на базе Chromium (Google Chrome, Microsoft Edge, Яндекс.Браузер и др.) или Firefox.
- Расширение [Tampermonkey](https://www.tampermonkey.net/) (или аналог: Greasemonkey, Violentmonkey).

## Установка

1. Установите расширение **Tampermonkey** из [Chrome Web Store](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) или [Firefox Add-ons](https://addons.mozilla.org/firefox/addon/tampermonkey/).
2. Нажмите на иконку Tampermonkey в панели инструментов и выберите **Создать новый скрипт**.
3. Удалите шаблонный код и вставьте содержимое файла [`kaitenlistchanger.js`](kaitenlistchanger.js).
4. Сохраните (`Ctrl+S` / `Cmd+S`).

Скрипт будет автоматически активироваться на всех поддоменах `https://kaiten.*/*`.

## Использование

1. Откройте любую карточку в Kaiten.
2. Захватите заголовок поля (надпись серым цветом) левой кнопкой мыши.
3. Перетащите поле в нужное место и отпустите.
4. Порядок сохранится автоматически.

## Технические детали

- Использует библиотеку [SortableJS](https://sortablejs.github.io/Sortable/) (подключается через `@require`).
- Данные хранятся в `GM_setValue` / `GM_getValue` — переносятся между поддоменами и не зависят от `localStorage`.
- Поиск полей основан на DOM-заголовках `<h6>` внутри контейнера `.cardContent`.
