Инструкция по установке пользовательского скрипта

Шаг 1. Установите расширение Tampermonkey
 1. Откройте браузер Google Chrome.
 2. Перейдите по ссылке: https://tampermonkey.net/?ext=dhdg&browser=chrome
 3. Нажмите “Установить”, следуйте инструкциям для установки расширения Tampermonkey.

Шаг 2. Добавьте пользовательский скрипт
 1. После установки Tampermonkey щёлкните по иконке расширения в правом верхнем углу браузера.
 2. Выберите “Создать новый скрипт”.
 3. Удалите шаблонный код и вставьте весь предоставленный код скрипта.
 4. Нажмите “Файл” → “Сохранить” или используйте сочетание клавиш Ctrl+S.

Шаг 3. Включите скрипт
 1. Убедитесь, что скрипт включён (переключатель рядом с его названием должен быть зелёным).
 2. При необходимости проверьте настройки Tampermonkey, чтобы убедиться, что скрипт активен.

Шаг 4. Включите режим разработчика в Chrome
 1. Перейдите в настройки Chrome:
 • Откройте меню (три точки в правом верхнем углу) → Дополнительные инструменты → Расширения.
 2. В правом верхнем углу включите переключатель “Режим разработчика”.
